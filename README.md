#An open source 2D game engine made in C# 
#
###Features

##### Platform
- Windows 10 / 8 / 7

##### API
- OpenGL 4+ / GLSL

##### Texture support
- JPG, PNG, BMP, PSD, GIF, HDR, PIC

##### Audio
- ogg

##### Fonts
- True Type Font Format (ttf)

##### Scene Features
- Scene management, extend predefined classes to implement your own scenes.
